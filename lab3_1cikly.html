Лабораторная работа Java Script 3.1 "Работа с циклами"

Задание №20

Дано целое число N(>0). Используя один цикл, найти сумму 1! + 2! + 3! +...+N! (выражение N!— N–факториал— обозначает произведение всех целых чисел от 1 до N: N!=1·2·...·N). Чтобы избежать целочисленного переполнения, проводить вычисления с помощью вещественных переменных и вывести результат как вещественное число.
